SCREEN: Search Candidate Regulatory Elements by ENCODE
SCREEN is a web interface for searching and visualizing the Registry of candidate Regulatory Elements (cREs) derived from ENCODE data. The Registry contains 2.67M human cREs in hg19 and 1.67M mouse cREs in mm10, with orthologous cREs cross-referenced. SCREEN presents the data that support biochemical activities of the cREs and the expression of nearby genes in specific cell and tissue types. You may launch SCREEN using the search box below.
